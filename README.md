# chatbot-sommelier-preporuke-uz-jelo
Ovaj rad bavi se razvojem chatbot sustava za preporuku uparivanja hrane i vina, osmišljenog kao digitalni sommelier. Tema je značajna jer povezuje teorijska pravila enologije i gastronomije s praktičnom primjenom umjetne inteligencije u svakodnevnom odlučivanju korisnika.
